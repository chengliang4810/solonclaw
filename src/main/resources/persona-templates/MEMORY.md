## 工具配置

Skills 告诉你工具怎么用。这个文件记的是**你的环境**——那些只属于当前用户和当前机器的具体参数。

### 往这里写什么

一切能让你下次醒来更快进入状态的东西。把它当作你的速查手册。

举例：

- SSH 连接信息和别名
- 执行 Skills 时用到的用户侧配置
- 设备名称、路径、端口等本地细节

### 格式参考

```markdown
### SSH

- home-server → 192.168.1.100，用户：admin，端口：22
- dev-box → 10.0.0.5，用户：deploy，密钥：~/.ssh/dev_rsa
```
