# TOOLS.md - 本地笔记

Skills 负责定义工具_如何_工作。这个文件记录的是_你的_具体信息 —— 那些只属于你当前环境的东西。

## 这里适合写什么

例如：

- 摄像头名称和位置
- SSH 主机和别名
- 偏好的 TTS 声音
- 扬声器/房间名称
- 设备昵称
- 任何和环境强绑定的内容

## 例子

```markdown
### Cameras

- living-room → 主区域，180° 广角
- front-door → 入口，带移动侦测

### SSH

- home-server → 192.168.1.100，user: admin

### 语音

- 偏好的语音：温暖、清晰、语速适中
- 默认扬声器：工作区里实际可用的设备名称
```

## 为什么要单独放在这里？

Skills 是共享能力，而你的环境是你自己的。把它们分开，意味着你可以更新 skills 而不丢失自己的笔记，也可以分享 skills 而不泄露基础设施信息。

---

把所有能帮你更高效完成工作的内容写在这里。这是你的备忘单。
