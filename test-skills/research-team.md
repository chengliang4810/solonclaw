# 研究团队协作技能

这是一个使用子 Agent 并行协作的示例技能。

## 功能描述

当你需要研究一个复杂主题时，可以启动多个子 Agent 并行工作：
- 一个子 Agent 负责收集基础信息
- 一个子 Agent 负责分析技术细节
- 一个子 Agent 负责总结要点
- 最后由主 Agent 汇总所有结果

## 使用方法

对 Agent 说：
```
请使用研究团队技能，并行研究以下主题：
"ReAct Agent 的工作原理"

请从以下角度研究：
1. 基础概念和定义
2. 技术实现细节
3. 应用场景和案例
```

## 内部机制

Agent 将会：
1. 创建 3 个子 Agent 并行工作
2. 每个子 Agent 负责一个研究角度
3. 子 Agent 完成后发送内部事件
4. 主 Agent 接收事件并汇总结果

## 预期输出

```
=== 研究报告：ReAct Agent 的工作原理 ===

## 1. 基础概念和定义
[子 Agent 1 的研究结果]

## 2. 技术实现细节
[子 Agent 2 的研究结果]

## 3. 应用场景和案例
[子 Agent 3 的研究结果]

## 总结
[主 Agent 的综合分析]
```

## 技术验证

此技能验证以下功能：
✓ 子 Agent 生成（spawn_subagent 工具）
✓ 并行执行（多个子 Agent 同时运行）
✓ 内部事件系统（任务完成通知）
✓ 结果汇总（主 Agent 整合子任务结果）
